Activity

Following the steps below, create a document in Microsoft Word applying the principles learnt in the training videos you watched on the previous page:

  1. Open Microsoft Word – this may be located as a shortcut on you toolbar, desktop or found via your program list
  2. Type the word ‘Monthly Report’  (press enter)

(the misspelling is deliberate for the purpose of this activity)

  1. Type the word ‘Month’ (press enter)
  2. Type  ‘Week 1’ (press enter)
  3. Type  ‘Week 2’ (press enter)
  4. Type  ‘Week 3’ (press enter)
  5. Type  ‘Week 4’ (press enter)
  6. Select the text you just typed in steps four to seven (Weeks 1 to Week 4)
  7. In the ‘home toolbar’ select the numbering to create a number list.
  8. Ensure the list is still selected.
  9. In the toolbar, select ‘line and paragraph spacing’ and change to 2.0
  10. Select the text ‘Monthly Report’
  • In the ‘home toolbar’ make the following changes:
    • Change the font to Arial
    • Change the font size to 14
    • Format as bold
  1. Conduct a check on spelling/grammar
  2. Click the ‘review toolbar’
  • Click ‘Spelling and Grammar’
  • Update any errors by reviewing the results
  • Select ‘Change’
  1. Click ‘File’
  • Save As
  • Save to a location such as ‘My Documents’
  • Rename the File ‘Monthly Report Template’.
