Operating Systems

Two of the most well-known operating systems are:

  • Microsoft Windows
  • macOS

Most common operating system for both home and work use. Windows 10 now includes inbuilt virus protection, which improves security, as updates can be initiated by Microsoft to PC’s in real time.

Pros
  • Lots of software and support available
  • Plenty of features for basic and advanced users
  • Hardware is more affordable
Cons
  • Hardware is much more expensive
  • Only runs on Apple products
