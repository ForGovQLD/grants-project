Types of software

There are three broad categories of software

Systems Software:

System software coordinates the activities and functions of hardware and software. It is the most basic type of software in any computer system, which is essential for other programs, applications and the whole computer system to function. Includes operating systems such as Microsoft Windows®, macOS®.

Application Software:

Application software run the tasks and functions the user requires the computer to perform. Includes programs such as Microsoft Office and Games. This is the software you most often use when you access computers.

Programming software:

Programming software is used to write, test, debug and develop other software programs and applications. This is used by programmers who write code to provide computers with instructions.
