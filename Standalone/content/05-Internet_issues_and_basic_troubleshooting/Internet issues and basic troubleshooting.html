Checking computer specifications and application requirements

When conducting work using a computer there may be a need to identify computer specifications to ensure applications are able to run effectively as per different software vendor requirements.

To do this in Microsoft Windows left mouse click on the ‘Start button’ and select ‘Settings’ (the gear icon).

When the new window is launched click on ‘System’ in the upper left corner and scroll down to, and select ‘About’, which provides important information about the processor size, speed, and the amount of installed RAM (memory).

Compare this information with the minimum specifications provided by the software vendor.
